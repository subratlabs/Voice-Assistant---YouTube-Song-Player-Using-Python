# 🎙️ Voice Assistant - YouTube Song Player

A Python-based voice assistant that listens to your voice commands and automatically opens YouTube to play the requested song. This project demonstrates the use of speech recognition, text-to-speech, and web automation to create a simple AI-powered voice assistant.

## 🚀 Features

* 🎤 Accepts voice commands through the microphone
* 🎵 Plays requested songs on YouTube
* 🌐 Automatically opens YouTube in the default web browser
* 🔊 Converts text responses into speech
* ⚡ Easy to use and beginner-friendly Python project

## 🛠️ Technologies Used

* Python 3
* SpeechRecognition
* pyttsx3
* PyWhatKit
* PyAudio
* WebBrowser

## 📦 Installation

1. Clone this repository:

```bash
git clone https://github.com/your-username/voice-assistant-youtube-player.git
```

2. Navigate to the project folder:

```bash
cd voice-assistant-youtube-player
```

3. Install the required libraries:

```bash
pip install -r requirements.txt
```

If you don't have a `requirements.txt` file, install the libraries manually:

```bash
pip install SpeechRecognition pyttsx3 pywhatkit PyAudio
```

## ▶️ How to Run

Run the Python file:

```bash
python main.py
```

Speak a command like:

* "Play Believer"
* "Play Shape of You"
* "Play Kesariya"

The assistant will automatically open YouTube and play the requested song.
